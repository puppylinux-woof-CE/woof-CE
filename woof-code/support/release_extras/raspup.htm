The first stable release of Raspup Puppy Linux is out!

Raspup Puppy is built from Raspbian Buster binary DEB packages, hence has binary compatibility with Raspbian and access to the Raspbian repositories. NOTE: It only runs on Raspberry Pi Boards!More comprehensive release notes and documentation of known issues are available.

Features include:

  • Latest bugfixes from upstream Raspbian Buster
  • Kernels following LTS Raspbian branches - 4.19.75
  • Many Icon and Graphical User Interface improvements throughout the system.
  • Ffmpeg and supporting programs including Pmusic and Mplayer.
  • Joe's Window Manager (JWM).
  • Visual and bugfix improvements to the tools to configure JWM, including PTheme, a global theme manager for JWM and ROX desktop
  • Updated Midori browser
  • Improvements to the graphical firewall program
  • Samba, used to share files with Windows, is shipped by default
  • pdf viewer
  • Comprehensive SFS Manager.

Raspup DISTRO_VERSION is shipped with a few differnet kernels to cover the Raspberry Pi Zero and 1, 2 and 3, and finally Pi 4. Find more information at the Raspup download page including a brief guide to which download is right for you.
